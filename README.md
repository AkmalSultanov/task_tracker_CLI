# Task Tracker CLI

This is a command-line tool for managing tasks using Python. It allows users to add, list, update, and delete tasks saved in a JSON file.

## How to Run

```bash
python task_tracker_CLI.py add "My Task"
python task_tracker_CLI.py list
python task_tracker_CLI.py delete 1
